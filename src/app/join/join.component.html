How to join us

Make sure you are running official Chia client 1.2.0 or later.

Connection

https://pool.openchia.io

Chia UI

  • Go to "Plots" menu in the left menu.
  • If you already have a Plot NFT select "Change Pool" then enter https://pool.openchia.io.
  • If you don't have a Plot NFT, add one, select "Connect to pool" and then enter https://pool.openchia.io.

Chia CLI

  • Create Plot NFT:
  • chia plotnft create -s pool -u https://pool.openchia.io
  • Verify plotnft and copy contract address:
  • chia plotnft show
  • Create plots using that plotnft:
  • chia plots create -c <contract address from above>
